01

市场事实

实体货架不缺吸尘器。
它缺的是一条被完整交付的任务链。

入表规则严格限定为零售商官网显示 Pickup Today / In Stock at Store Today 的商品。线上可购、Ship to Store、Online only 不进入这一判断。

货架的真实分工

每个品类都有购买理由，
但没有一个完整解决“开始—完成—归位”。

  1. 01低价湿干

    即刻处理车库脏污和小型液体；代价是搬机、接电、拖管、收附件。

  2. 02家庭立式

    宠物、地毯、HEPA 和全屋地面；不覆盖湿污和车库重活。

  3. 03无线 / 电池平台

    轻便、随手、已有工具用户；容量、续航和整车覆盖有限。

  4. 04局部“少收尾”

    例如收电源线；仍未解决工作软管、附件和归位流程。

02

用户摩擦

用户缺的不是更多吸力，
而是更少的准备与收尾。

以下比例来自可公开读取的负面 / 求助信号的多标签估算：Walmart Hyper Tough（796 条评论）、Lowe’s BISSELL 35332（约 445 条）、Shark NV366（198 条）、Home Depot Milwaukee 0970-20（3,930 条）及 Eureka 系列公开评论摘要、标签与 Q&A。单条内容可命中多项，比例不相加，不等同于全体用户不满意率。

准备

够不着，也不想搬

≈28% 指向软管 / 电源线不够到位；“轻便”和“大容量”被迫二选一。

机会：固定主机承载容量，41FT 覆盖承载可达。
清洁

清到深处，能力不稳定

≈21% 指向真实材质上的刷头 / 吸力表现；高绒地毯操作阻力也频繁出现。

机会：通尘通电把电动刷和吸力交付到末端。
收尾

“吸完”并不等于完成

≈31% 指向线、软管、附件、滤芯与归位的连续摩擦，是最强共性信号。

机会：自动回收和附件固定座必须成为可见结果。

因此

新品若只多一个功能，仍会落入既有货架的单点竞争。它必须让“准备—清洁—收尾”成为同一条体验。

03

我们有什么

把产品参数，翻译成用户
可以感知的一次完成。

依据《尘电吸尘器整合营销上市规划》：首发只围绕车辆、车库、工作台连续重活；未来工具生态不是首发承诺。

尘电新品的价值不在“壁挂”本身，而在于把 容量、覆盖、动力末端与回收 变成一个连续系统。

固定主机+40/41FT 覆盖+电动末端+自动回收
01

20L 湿干主机

容量留在墙面，用户不用把桶机搬到脏污前；解决“轻便或容量二选一”。

02

40/41FT 固定覆盖

从一个墙点到车辆、地面、工作台和门口区；需要用不同车库户型实测证明。

03

EnduraPower 通尘通电

在同一可回缩尘管内把吸力和电力送至动力滚刷、带灯缝隙吸等末端，解决深洁结果。

04

SyncRetract 自动回收

把搬、接、拖、盘、收压缩成拉出、使用、回收；耐久和卡管率必须用可靠性证据支撑。

机会定义

不要以“壁挂吸尘器”进入市场。
要以“车库连续重活清洁工作站”进入市场。

市场已有：低价湿干、宠物地面、无线随手与工具电池平台。

市场缺少：用户从固定点出发，不搬机、不换插座，完成深洁并自动归位。

尘电的占位：用通尘通电 + 自动回收，交付一次任务结果而非堆叠参数。

04

上市验证

先证明“为什么值 $549.99”，
再决定“在哪里放量”。

一个进入主张

看见重污，拉出就能深洁；
完成以后，系统自己回到该在的位置。

首发只承诺车辆 / 车库 / 工作台连续重活，不用未来生态替代当前交付。
证据 01

覆盖是否真实

双车位车库 + SUV + 工作台，从一处墙点完成最远端、门和转角任务。

看：到达率 / 安装成功率
证据 02

深洁是否可见

脚垫砂砾、座椅织物、宠物毛、暗缝；必须展示“先刷松，再吸走”的单次结果。

看：前后对比 / 完成时间
证据 03

回收是否可信

回收顺畅、卡管率、循环寿命、湿干维护、噪音、关键备件时效。

看：循环测试 / 退货归因

进入顺序不是渠道顺序，而是证据顺序

每一步都要消除一种购买风险。

  1. 01问题验证

    收集“为什么强力工具总是太麻烦”的真实原话，确认首发任务与人群。

  2. 02品类讲清

    用实机把“通尘通电”讲成一眼可见的任务结果，而非参数名词。

  3. 03预售验价

    以安装教育、FAQ、真实演示和两档 SKU 验证理解、支付与异议。

  4. 04规模放量

    仅在产品可靠、转化与服务证据均成立后，扩展零售与附件复购。

管理层只需盯住三道门

产品能交付吗？

覆盖、末端性能、回收寿命、安装成功率和备件可得性有无量化证据。

用户愿意付费吗？

3 秒能否理解主张；预售转化、取消原因和价格异议是否支持 $549.99。

规模能承接吗？

内容表达、Owner Hub、售后、耗材和退货归因能否形成闭环。
